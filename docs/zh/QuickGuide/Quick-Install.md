# 快速安装

选择适合你的安装方式：

## 二进制运行（推荐）

直接下载预编译包，一键启动，支持单机和集群模式。

→ [二进制运行指南](../InstallationDeployment/Binary-Deployment.md)

## Docker

拉取官方镜像，无需本地安装 Rust 环境。

→ [Docker 部署指南](../InstallationDeployment/Docker-Deployment.md)

## Kubernetes

通过 Operator 部署到 K8S 集群。

→ [K8S 部署指南](../InstallationDeployment/Kubernetes-Operator.md)

## 从源码构建

如果没有现成二进制包，或需要修改代码，可以本地编译。

→ [编译打包指南](../ContributionGuide/ContributingCode/Build-and-Package.md)
