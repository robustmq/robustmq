# Quick Install

Choose the installation method that fits your needs:

## Binary (Recommended)

Download a pre-built package and start immediately. Supports both standalone and cluster mode.

→ [Binary Deployment Guide](../InstallationDeployment/Binary-Deployment.md)

## Docker

Pull the official image — no local Rust toolchain required.

→ [Docker Deployment Guide](../InstallationDeployment/Docker-Deployment.md)

## Kubernetes

Deploy to a K8S cluster using the Operator.

→ [K8S Deployment Guide](../InstallationDeployment/Kubernetes-Operator.md)

## Build from Source

If no pre-built binary is available, or you need to modify the code, build locally.

→ [Build and Package Guide](../ContributionGuide/ContributingCode/Build-and-Package.md)
